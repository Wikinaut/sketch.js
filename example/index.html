Sketch.js examples:

1. Simple usage:

Code:

$(document).ready(function() {
  $('#canvas1').sketch();
});

2. Color change:

Code:

$('#canvas2').sketch();
$('.color-btn').click(function(e) {
  e.preventDefault();

  $('#canvas2').sketch('set', 'color', $(this).data('color'));
});

3. Size change:

Choose size:

Code:

$('#canvas3').sketch();
$('.change-size').change(function(e) {
  $('#canvas3').sketch('set', 'size', $(this).val());
});

4. Custom background and save:

Save sketch

The following example of save won't work on local machines, due to cross-origin policy. To try out this feature, upload that page to your web-server.

Code:

$('#canvas4').sketch();
$('#canvas4').sketch('loadBg', 'img.jpg');
$('.save-sketch').click(function(e) {
  e.preventDefault();
  $('#canvas4').sketch('download');
});

5. Resize handling:

Resize sketch

This example shows, that your drawings are resized proportionally to your canvas, when it resizes.
Not as in original intridea/sketch.js library.

Code:

$('#canvas5').sketch();
$('.resize-sketch').click(function(e) {
  e.preventDefault();
  $('#canvas5').height(
    Math.floor(Math.random() * (400 - 100)) + 100
  );
});

6. Undo method:

Undo last action

Code:

$('#canvas6').sketch();
$('.undo-sketch').click(function(e) {
  e.preventDefault();
  $('#canvas6').sketch('undo');
});
